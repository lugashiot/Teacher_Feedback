Teacher Feedback (HTLinn CCA Anichstraße HTBLuVA)

Unser Tool ermöglicht allen Schülern, ihren Lehrern ein anonymes Feedback zu geben, ohne Angst, dass dieses Einfluss auf seine Noten oder die Haltung des Lehrers auf den Schüler hat.

Für Schüler:

Sie können ihrer Lehrperson erst nach erhalten einer E-Mail ein Feedback geben. Das Versenden dieser Mail wird von der jeweiligen Lehrperson veranlasst.

Für Lehrer:

Unter http://www.htlinn.tech/login/ können Sie sich mit den, einmalig per Mail erhaltenen, Zugangsdaten anmelden. Im Dashboard können Sie dann auswählen an welche ihrer Klassen eine Feedback Mail versendet werden soll.
